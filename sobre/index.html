Sobre

A Radialle é um grupo de estudos em tecnologia da informação criado e mantido pela RapidLight. Sua missão é contribuir com a comunidade através do compartilhamento de conhecimentos e desenvolvimento de projetos abertos com licenças permissivas.

Experimentos

Pequenos componentes de hardware ou software criados por integrantes da Radialle para fins de aprendizado, demonstração ou diversão. Não são ativamente mantidos e tampouco destinados a uso em ambientes de produção.

  • catfinder
    Aplicação Android que identifica fotos de gatos no dispositivo utilizando o ML Kit.
    https://github.com/radialle/catfinder
  • ct-locate
    Ferramenta que se conecta a um Chromecast e tenta descobrir sua localização física através da lista de redes Wi-Fi próximas e a API de geolocalização da Google.
    https://github.com/radialle/cast-toolkit/blob/master/ct-locate.py
  • mfc-pn532
    Imagem Docker para recuperar chaves de cartões MIFARE Classic usando o módulo RFID PN532 conectado ao computador via adaptador USB para UART.
    https://github.com/radialle/mfc-pn532
  • pdmtools
    Demonstração de integração entre Python e D-Bus com uso da biblioteca pydbus.
    https://github.com/radialle/pdmtools
  • rain
    Fork do Telegram para Android.
    https://github.com/radialle/rain
  • redbridge
    Ferramenta para identificação de servidores ADB (Android Debug Bridge) publicamente acessíveis.
    https://github.com/radialle/redbridge
Última atualização em 7 de abril de 2020.
